# Studentfoto's

Plaats hier de pasfoto's van de studenten.

## Naamconventie

Gebruik de volledige naam in kleine letters, spaties vervangen door koppeltekens, eventuele bijzondere tekens vereenvoudigd:

```
voornaam-achternaam.jpg
```
